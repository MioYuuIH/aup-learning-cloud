# Deep Learning Labs

We provide a brief overview of the labs included in the `DL` directory. Each lab focuses on a specific aspect of deep learning and machine learning, offering hands-on experience with key concepts and techniques from foundational algorithms to advanced neural architectures.

## Lab Descriptions

### **DL01: Principal Component Analysis (PCA)**
- **Dataset**: Wine dataset.
- **Focus**: Dimensionality reduction and data visualization.
- **Key Learning**: Mathematical foundations of PCA, eigendecomposition, data preprocessing, and standardization techniques.
- **Implementation**: PCA from scratch using eigendecomposition.

### **DL02: Support Vector Machine (SVM)**
- **Focus**: Classification using support vector machines.
- **Key Learning**: Understanding SVM concepts, kernel methods, and classification boundaries.
- **Implementation**: SVM implementation and optimization.

### **DL03: K-Means Clustering**
- **Focus**: Unsupervised learning and clustering algorithms.
- **Key Learning**: Centroid-based clustering, convergence criteria, and cluster validation.
- **Implementation**: K-means algorithm from fundamentals.

### **DL04: Decision Tree**
- **Focus**: Tree-based classification and regression.
- **Key Learning**: Information gain, entropy, pruning techniques, and interpretable ML models.
- **Implementation**: Decision tree construction and evaluation.

### **DL05: Regression Model**
- **Focus**: Linear and polynomial regression techniques.
- **Key Learning**: Statistical modeling, feature engineering, and predictive analytics.
- **Implementation**: Various regression approaches and model evaluation.

### **DL06: Neural Network**
- **Focus**: Building neural networks from scratch.
- **Key Learning**: Forward propagation, backward propagation, parameter updates, binary and multi-class classification.
- **Implementation**: Complete neural network implementation with linear layers and activation functions.

### **DL07: Word2Vec**
- **Dataset**: Text8 corpus
- **Focus**: Word embeddings and natural language processing.
- **Key Learning**: Vector representations of words, skip-gram model, and semantic relationships.
- **Implementation**: Word2Vec training and visualization using t-SNE.

### **DL08: Basic CNN Tasks**
- **Dataset**: CIFAR-10
- **Focus**: Convolutional Neural Networks for image classification.
- **Key Learning**: Convolution layers, pooling operations, CNN architecture design, and image preprocessing.
- **Implementation**: CNN construction, training loops, and model evaluation.

### **DL09: AutoEncoder**
- **Focus**: Unsupervised learning and data reconstruction.
- **Key Learning**: Encoder-decoder architecture, latent space representation, and dimensionality reduction.
- **Implementation**: AutoEncoder for data compression and reconstruction.

### **DL10: Sequence-to-Sequence (Seq2Seq)**
- **Dataset**: English-Chinese translation pairs.
- **Focus**: LSTM-based sequence modeling.
- **Key Learning**: Encoder-decoder architecture, LSTM networks, teacher forcing, and sequence translation.
- **Implementation**: English-to-Chinese translation model.

### **DL11: Generative Adversarial Networks (GANs)**
- **Dataset**: MNIST
- **Focus**: Deep Convolutional GAN (DCGAN) implementation.
- **Key Learning**: Generator and discriminator networks, adversarial training, and image generation.
- **Implementation**: DCGAN in PyTorch for synthetic image generation.

### **DL12: Transformer**
- **Focus**: Attention mechanisms and transformer architecture.
- **Key Learning**: Self-attention, positional encoding, and modern NLP architectures.
- **Implementation**: Mini transformer from scratch for text generation.

## Lab Organization

The labs are organized progressively from foundational machine learning concepts to advanced deep learning architectures:

**Foundational ML (DL01-DL05)**: Classical machine learning algorithms including PCA, SVM, K-Means, Decision Trees, and Regression models that provide essential background for understanding deep learning.

**Core Deep Learning (DL06-DL09)**: Fundamental neural network concepts including building networks from scratch, word embeddings, CNNs, and autoencoders.

**Advanced Architectures (DL10-DL12)**: State-of-the-art models including sequence-to-sequence networks, GANs, and transformers, representing the cutting edge of deep learning research.
